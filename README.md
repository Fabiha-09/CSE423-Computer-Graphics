# CSE423-Computer Graphics

This repository contains the code and examples for the Computer Graphics course (CSE423).

Contents:
- OpenGL Python examples (Hello_openGL.py, Lets_draw_sth.py)
- `OpenGL/` library files

Quick start (Windows):

1. Ensure Python 3.8+ is installed. Use the `py` launcher if available.
2. (Optional) Create and activate a virtual environment:

```powershell
py -3 -m venv .venv
.\.venv\Scripts\Activate.ps1
```

3. Install dependencies (if any):

```powershell
pip install PyOpenGL PyOpenGL_accelerate
```

4. Run an example:

```powershell
py Hello_openGL.py
```
